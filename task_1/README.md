## Title
GitHub, Virtual Environment, and Linux Basics

## Description
This taks folder is for demonstration of setup of venv and usage of basic linux commands

## Setup Instructions
1.) clone the repository `git clone https://github.com/Muizzahmed786/Synergy_TP.git`

## venv setup
1.) in root project folder run this command to install venv `python -m venv venv`
2.) to activate venv `venv\Scripts\activate`

## to install requirements
run this command inside Task_1 folder `pip install -r requirements.txt`

## To run python file
run this command from the project root i.e Synergy_TP `python task_1/src/hello.py`
